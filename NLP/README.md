# Natural Language Processing (NLP)

## Overview

This folder contains my learning and practice work related to Natural Language Processing (NLP).

NLP is a field of Artificial Intelligence that enables computers to understand, process, and generate human language.

## Topics Covered

### Text Preprocessing

* Lowercasing
* Removing Punctuation
* Removing Stop Words

### Tokenization

* Word Tokenization
* Sentence Tokenization

### Stemming

* Porter Stemmer
* Snowball Stemmer

### Lemmatization

* WordNet Lemmatizer

### Encoding Techniques

* One-Hot Encoding
* Bag of Words (BoW)
* TF-IDF
* N-Grams

## Tools & Libraries

* Python
* NLTK
* Scikit-learn
* Pandas
* NumPy

## Files

| Topic            | Description                        |
| ---------------- | ---------------------------------- |
| One_Hot_Encoding | Converts words into binary vectors |
| Bag_of_Words     | Creates word frequency vectors     |
| TF_IDF           | Calculates term importance         |
| N_Grams          | Generates word sequences           |

## Learning Objective

The goal of this folder is to build a strong foundation in NLP and understand how text data is converted into numerical features for Machine Learning models.
